# [1008] 출력하기 08
> ┌┬┐ <br/>
  ├┼┤ <br/>
  └┴┘ <br/>

```java
public class Main {
    public static void main(String[] args){
         System.out.println("\u250C\u252C\u2510\n\u251C\u253C\u2524\n\u2514\u2534\u2518");
    }
}
```
유니코드로 제출하라는 지문을 못보고 특수문자로 입력했다가 에러..<br/>
'\u' 서식문자열을 사용하면 유니코드를 출력할 수 있다.
