# [1258] 1부터 n까지 합 구하기
> **[입력]** 100<br/>
  **[출력]** 5050

```java
import java.util.Scanner;
public class Main {
   public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      int sum = 0;
      int input = sc.nextInt();
      for(int i = 0; i <= input; i++) {
         sum += i;
      }
      System.out.println(sum);
   }
}
```

# [1266] n개의 수의 합
> **[입력]** <br/>
5<br/>
3 5 7 7 2<br/>
  **[출력]** 24

```java
import java.util.Scanner;
public class Main {
   public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      int sum = 0;
      int input = sc.nextInt();
      for(int i = 0; i < input; i++) {
         sum += sc.nextInt();
      }
      System.out.println(sum);
   }
}
```
